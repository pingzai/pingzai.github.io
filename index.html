HTML/CSS

发表于 2017-03-21

一、标签分类：

       1.行内标签：
        span,a,i,em,b,strong,
        1)共处一行
        2)不支持宽高
        3)受空格和回车影响
    2.块元素
        p,div,h1-h6,ul,ol,dl,li,dt,dd
        1)独占一行
        2)支持宽高
        3)不受空格和回车影响
    3.行内块
        img,input
        1)共处一行
        2)支持宽高
        3)受空格和回车影响
元素转换：
    1)行内元素转成块元素    display:block;
    2)块元素转成行内元素    display:inline;(尽量不用)
    3)转行内块            display:inline-block;
书写规范：
    1)双标签必须闭合
    2)单标签合理闭合(/)
    3)合理嵌套和缩进
        a.块元素可以包块和行内(p只能包行内，不能包块)
        b.行内元素可以包行内(不要包自己)*(a可以包块,一般把a转化成块)*
    4)所有字母都小写

发表于 2017-03-21

50.预解析:

会预先解析一些(变量定义、函数会预解析);
var a=12;  
    只把var a提到作用域的顶部
js会把变量的声明(仅仅是声明)提到顶部。不会突破作用域

执行步骤:
    读取一个script标签>预解析>执行里面的代码>
    读取下一个script标签>预解析>执行里面的代码
    不会突破script标签
undefined和not defined   
    undefined存在没有赋值
alert(aaa);        var aaa=12;
    not defined  没定义 直接用
alert(aaa);        aaa=12;

预解析作用范围：
会找本作用域之内(函数、script)

函数也有预解析,但是要注意:
当以变量的形式声明的函数,预解析会按照变量的特性走
//var aaa;
aaa();
var aaa=function () {
   alert(123)
}        //undefined;
