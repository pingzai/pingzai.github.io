2.函数声明和调用

声明:function 函数名() {
        //code
    }
    调用:函数名()

3.事件: 任何标签都能加事件 操作谁给谁加事件

点击事件:onclick
鼠标移入:onmouseover
鼠标移出:onmouseout
鼠标按下:onmousedown
鼠标抬起:onmouseup
select下选中的

4.document.body能直接选中body

**5.if判断–流程控制语句
if（条件）{语句1}； 简写：条件&&语句1
if（条件）{语句1}else if（条件）{语句2}; 简写：条件？语句1：语句2；（三目运算符）

switch（条件）{
case 情况1：
语句
break；
case 情况2：
语句
break;
default：
语句
break；
}

**break； 中断本次执行
for(var i=0;i<10;i++){ if(i==3){ break; //中断本次循环}
alert(i); //0,1,2
}

**continue； 跳过本次执行
for(var i=0;i<10;i++){ if(i==3){ continue; //跳出本次循环}
alert(i);//0,1,2,4,5,6,7,8,9
}
= 赋值 == 判断比较 === 全等

6.style 玩的都是行间

**7.传参:
(一个参数) 声明:function 名字(参数){
//code
//参数
}
调用:名字(参数)

(多个参数) 声明:function 名字(参数1,参数2){
//code
//参数1,参数2
}
调用:名字(参数1,参数2)

8.标示符和表达式:

9.获取元素

document.getElementById(‘id’)
通过class选取一组元素
document.getElementsByClassName(‘class’)
